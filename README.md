## Introduction

This project is a collection of different graphical applications built using SDL. The aim is to explore the capabilities of SDL and create visually appealing and interactive programs.

## Installation

To get started with the project, follow these steps:

1. **Clone the repository**:
    ```sh
    git clone https://github.com/Bridiro/c_sdl_geometry.git
    cd c_sdl_geometry
    ```

2. **Install dependencies**:
    Ensure you have SDL2 and SDL2_ttf installed on your system. You can download it from the [SDL github](https://github.com/libsdl-org/SDL/releases/tag/release-2.30.8).

3. **Build the project**:
    ```sh
    make
    ```

## Usage

Run the compiled executable to start the application:
```sh
./c_sdl_geometry
```
or by using
```sh
make run
```
